Enhancements
These originated as Greasemonkey User Scripts and have been ported over to this Chrome extension. The code has been reworked some, but for the most part the original authors' code remains intact.

    Footprints
  • biggerDescription.js - Increases the size of the previous description area in edit-ticket mode. Also includes the functions from largerAssignees.js, which increases the size of the assignees section on the ticket page.
  • quickAssign.js - Adds buttons to the side of the assingees box to allow assigning of tickets with one click. Note: not currently working :(
  • userSearch.js - Adds a search bar under the normal Footprints search bar that allows for search all tickets by username.
  • tcenterLove.js - Adds a link to jump to a TCenter ticket if one exists.
    • Wiki
  • kickassWiki.js - A collection of JavaScript functions and CSS classes that make the Confluence wiki a little bit more kickass.
    • CLAWS
  • clawsTools.js - A collection of functions that makes searching the dreaded CLAWS into a breeze on a summer morning.

Links

I collected these links from the Desktop Support wiki page, which lists a few "quick links" on the front page. In the future, I'll code a feature that lets you add your own links, but for now this is what you get.

If you really want to add some links to there, I suggest you git clone on GitHub and edit the /data/links.json file, which is where the data is pulled from. Then, submit a pull request and I'll merge your changes.

Notepad

I'll admit, this feature is experimental. Every time you stop typing, it saves what you wrote into a JavaScript object, and then stores that in Chrome's cloud storage. The Chrome cloud is associated with whatever account you are logged into Chrome with. Personally, I created a separate Chrome account with my @rit.edu username; this stores all of my bookmarks, history, and SuperTech notes accross every station I log into.

Since this is experimental, I don't recommend you keep anything super-important in the Notepad. Also, since this goes to Google's servers, you probably shouldn't put anythign super-confidential in there either. But, when you're surfing around looking for a solution to a problem and need to just jot down a few notes, this becomes quite handy.
